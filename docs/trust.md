# Trust & Safety

## Why does Windows show a SmartScreen warning?

SCStats is currently unsigned. Windows SmartScreen warns users about new
applications with low download reputation.

This does not indicate malicious behavior.

---

## What SCStats accesses

- Local Star Citizen log files only
  - `Game.log`
  - `logbackups`

---

## What SCStats does NOT do

- No memory access
- No injection or hooks
- No network communication
- No data uploads

All analysis is performed locally on your machine.
